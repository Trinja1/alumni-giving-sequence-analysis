# Alumni Giving Sequence Analysis — R / TraMineR

## Project Overview
Longitudinal giving analysis of approximately 3,200 Northeastern
University alumni (graduating classes 1990–2006) with significant
vendor wealth ratings. Performed during the 10-year Empower
fundraising campaign (FY2008–FY2017).

## Methodology
Sequence analysis and sequence clustering performed using the
TraMineR package in R. Giving states defined across 7 tiers
ranging from $0 to $1M–$9.99M.

## Cluster Results
Four donor behavior archetypes identified:
- Type 1 (n=1,886) — Predominantly non-donors, sporadic low-level giving
- Type 2 (n=915)   — Non-giving with emerging activity in later years
- Type 3 (n=234)   — Consistent low-to-mid level engagement
- Type 4 (n=211)   — Sustained giving across full campaign period
                     at higher gift tiers — primary major gift prospects
